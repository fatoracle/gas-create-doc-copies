Enter document Id below of the Google Doc you want to make copies of and number of copies you want to make.

If no docId specified, the current document will be copied.

If number of copies not specified, 5 copies will be made.

This was developed as an internal tool used by the JoeZoo Test team to create test documents for the JoeZoo Express Google Docs add-on. Code for this add-on is available at the github link below. Go forth and fork.

Source code: https://github.com/galtroarc/gas-create-doc-copies
